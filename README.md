# Laboratório do Caos

## Sobre o Projeto

Laboratório do Caos é um jogo 2D desenvolvido em Python utilizando a biblioteca Pygame.

O jogador controla um cientista que ficou preso dentro de um laboratório após um experimento falhar e causar uma série de anomalias físicas.

Para escapar, será necessário atravessar diferentes setores do laboratório, superar obstáculos e utilizar conceitos de Física como gravidade, energia, força e eletricidade.

---

## História

Durante um experimento envolvendo manipulação das leis da Física, um acidente provoca uma falha em todo o complexo científico.

Portas são bloqueadas, sistemas entram em colapso e diversos setores tornam-se perigosos.

O cientista sobrevivente precisa explorar o laboratório, coletar cartões de acesso e desligar a máquina responsável pelo acidente antes que todo o sistema entre em colapso.

---

## Objetivo

Explorar o laboratório.

Coletar itens importantes.

Desbloquear portas.

Escapar do complexo científico.

---
